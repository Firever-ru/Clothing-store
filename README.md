Онлайн магазин в Телеграмм боте на Python,библиотека aiogram 2.25.1, SQLite. Реализовано 2 меню - владелец и покупатель. В меню владельца можно выкладывать товар, изменять размеры, цену, фото. Смотреть статус отправок. Реализована система отзывов,после получения заказа клиент может написать отзыв, который будет продублирован в телеграмм канале.
